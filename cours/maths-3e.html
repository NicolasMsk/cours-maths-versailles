1. Calcul littéral & identités remarquables

Au programme

  • Développer et réduire
  • Factoriser, reconnaître une différence de carrés
  • Maîtriser les trois identités remarquables

Les identités remarquables se lisent dans les deux sens : de gauche à droite pour développer, de droite à gauche pour factoriser.

Formules clés

  • $(a+b)^2 = a^2 + 2ab + b^2$
  • $(a-b)^2 = a^2 - 2ab + b^2$
  • $(a+b)(a-b) = a^2 - b^2$

Méthode

  1. Identifier $a$ et $b$
  2. Appliquer l’identité correspondante
  3. Vérifier en développant mentalement

Exemple 1

$(3x + 2)^2 = (3x)^2 + 2(3x)(2) + 2^2 = 9x^2 + 12x + 4$.

Exemple 2

$16x^2 - 9 = (4x)^2 - 3^2 = (4x - 3)(4x + 3)$.

2. Équations & équations produit

Au programme

  • Résoudre une équation du premier degré
  • Résoudre une équation produit nul
  • Mettre en équation un problème

Un produit de facteurs est nul si, et seulement si, l’un au moins de ses facteurs est nul.

Formules clés

  • $A \times B = 0 \iff A = 0 \text{ ou } B = 0$

Méthode

  1. Tout ramener du même côté (égal à 0)
  2. Factoriser si possible
  3. Annuler chaque facteur

Exemple 1

$5x - 7 = 2x + 8 \Rightarrow 3x = 15 \Rightarrow x = 5$.

Exemple 2

$(x - 3)(2x + 1) = 0 \Rightarrow x = 3$ ou $x = -\dfrac{1}{2}$.

3. Théorème de Pythagore et réciproque

Au programme

  • Calculer une longueur
  • Utiliser la réciproque pour prouver un angle droit
  • Calculer une valeur approchée

Le théorème permet de calculer une longueur dans un triangle rectangle ; la réciproque permet de démontrer qu’un triangle est rectangle.

Formules clés

  • Théorème : rectangle en $A \Rightarrow BC^2 = AB^2 + AC^2$
  • Réciproque : $BC^2 = AB^2 + AC^2 \Rightarrow$ rectangle en $A$

Méthode

  1. Identifier le plus grand côté (hypoténuse potentielle)
  2. Calculer $BC^2$ et $AB^2 + AC^2$
  3. Comparer : égaux ⇒ rectangle

Exemple 1

$AB = 6$, $AC = 8$ : $BC^2 = 36 + 64 = 100$, donc $BC = 10$.

Exemple 2

Côtés $5$, $12$, $13$ : $5^2 + 12^2 = 169 = 13^2$, donc le triangle est rectangle.

4. Théorème de Thalès

Au programme

  • Écrire les rapports égaux
  • Calculer une longueur
  • Reconnaître la configuration (et la réciproque)

Dans une configuration de Thalès avec $(MN) \parallel (BC)$, les côtés des deux triangles sont proportionnels.

Formules clés

  • $\dfrac{AM}{AB} = \dfrac{AN}{AC} = \dfrac{MN}{BC}$

Méthode

  1. Repérer les droites parallèles et les points alignés
  2. Écrire la triple égalité des rapports
  3. Utiliser le produit en croix pour la longueur cherchée

Exemple 1

$AM = 3$, $AB = 8$, $BC = 10$ : $MN = 10 \times \dfrac{3}{8} = 3{,}75$.

Exemple 2

$AM = 4$, $AB = 6$, $AN = 6$ : $AC = \dfrac{6 \times 6}{4} = 9$.

5. Fonctions linéaires et affines

Au programme

  • Distinguer linéaire et affine
  • Calculer image et antécédent
  • Déterminer une fonction, lire un graphique

Une fonction linéaire $f(x) = ax$ modélise une situation de proportionnalité ; une fonction affine $f(x) = ax + b$ a une représentation en droite.

Formules clés

  • Linéaire : $f(x) = ax$
  • Affine : $f(x) = ax + b$
  • Image de $x$ : $f(x)$ ; antécédent de $y$ : $x$ tel que $f(x)=y$

Méthode

  1. Pour une image : remplacer $x$ par sa valeur
  2. Pour un antécédent : résoudre $f(x) = y$

Exemple 1

$f(x) = 3x - 5$ : $f(4) = 12 - 5 = 7$, et l’antécédent de $7$ vérifie $3x - 5 = 7$ d’où $x = 4$.

Exemple 2

$g$ linéaire avec $g(2) = 10$ : $a = \dfrac{10}{2} = 5$, donc $g(x) = 5x$ et $g(5) = 25$.
